## 🎤 HR Interview Practice

### How It Works

The HR Interview feature helps you practice common interview questions with real-time feedback.

### Step-by-Step Guide

1. **Navigate to HR Interview**
   - Click "HR Interview" in the navigation menu
   - You'll see the interview interface

2. **Setup Your Environment**
   - Find a quiet, well-lit space
   - Position yourself in front of the camera
   - Test your microphone

3. **Start Interview**
   - Click "Start Interview" button
   - Allow camera and microphone access
   - Wait for the first question

4. **Answer Questions**
   - Read the question carefully
   - Think before you speak
   - Click "Start Recording" to begin
   - Speak clearly and naturally
   - Click "Stop Recording" when done

5. **Review Feedback**
   - AI analyzes your answer
   - Emotion detection shows your expressions
   - Get suggestions for improvement
   - Compare with ideal answers

6. **Continue Practice**
   - Click "Next Question" for more practice
   - Track your progress
   - Review all feedback at the end

### Common HR Questions

The platform includes these popular questions:

1. **Tell me about yourself**
   - Duration: 60-90 seconds
   - Focus: Professional background, skills, goals
   - Tip: Keep it relevant to the job

2. **Where do you see yourself in 5 years?**
   - Duration: 45-60 seconds
   - Focus: Career goals, growth mindset
   - Tip: Show ambition but be realistic

3. **What are your greatest strengths?**
   - Duration: 45-60 seconds
   - Focus: 2-3 key strengths with examples
   - Tip: Relate to job requirements

4. **What is your greatest weakness?**
   - Duration: 30-45 seconds
   - Focus: Real weakness + improvement plan
   - Tip: Show self-awareness and growth

5. **Why do you want to work here?**
   - Duration: 45-60 seconds
   - Focus: Company research, alignment with values
   - Tip: Be specific and genuine

### Tips for Best Results

✅ **Do:**
- Maintain eye contact with camera
- Smile naturally
- Speak at a moderate pace
- Use specific examples
- Show enthusiasm
- Be authentic

❌ **Don't:**
- Read from notes
- Speak too fast or too slow
- Give generic answers
- Appear disinterested
- Interrupt the question
- Ramble without structure

### Understanding Emotion Detection

The system tracks these emotions:
- **Happy** 😊 - Shows enthusiasm and positivity
- **Neutral** 😐 - Professional and composed
- **Surprised** 😲 - Shows engagement
- **Sad** 😢 - May indicate lack of confidence
- **Angry** 😠 - Avoid showing frustration
- **Fearful** 😨 - Work on building confidence
- **Disgusted** 🤢 - Maintain positive expressions

**Ideal Emotion Mix:**
- 60-70% Neutral (professional)
- 20-30% Happy (enthusiastic)
- 10% Surprised (engaged)

## 📄 Resume ATS Scoring

### What is ATS?

ATS (Applicant Tracking System) is software used by companies to filter resumes. Our tool helps you optimize your resume to pass these systems.

### How to Use

1. **Navigate to Resume ATS**
   - Click "Resume ATS" in the menu
   - You'll see the upload interface

2. **Prepare Your Resume**
   - Save as PDF or DOCX
   - Ensure text is selectable (not image)
   - Use standard fonts
   - Keep formatting simple

3. **Upload Resume**
   - Click "Choose File"
   - Select your resume
   - Enter job description or role
   - Click "Analyze"

4. **Wait for Analysis**
   - Processing takes 10-30 seconds
   - System extracts text
   - Analyzes multiple factors
   - Generates detailed report

5. **Review Results**
   - Overall ATS Score (0-100%)
   - Hard Skills Analysis
   - Soft Skills Analysis
   - Structure Score
   - Word Count Analysis
   - Grammar Corrections

### Understanding Your Score

**90-100%** - Excellent! 🎉
- Resume is well-optimized
- Strong keyword match
- Good structure and format
- Ready to apply

**70-89%** - Good 👍
- Solid foundation
- Some improvements needed
- Add missing keywords
- Fix minor issues

**50-69%** - Needs Work 📝
- Significant gaps
- Missing key skills
- Structure issues
- Follow suggestions carefully

**Below 50%** - Major Revision Needed 🔧
- Substantial rewrite required
- Missing critical elements
- Poor ATS compatibility
- Consider professional help

### Score Breakdown

**Hard Skills (40%)**
- Technical skills
- Tools and technologies
- Programming languages
- Certifications
- Domain expertise

**Soft Skills (20%)**
- Communication
- Leadership
- Teamwork
- Problem-solving
- Adaptability

**Structure (20%)**
- Proper sections
- Clear headings
- Logical flow
- Contact information
- Professional summary

**Word Count (20%)**
- Optimal: 400-600 words
- Too short: < 300 words
- Too long: > 800 words

### Optimization Tips

**Keywords:**
- Use exact terms from job description
- Include industry-standard terminology
- Add relevant acronyms
- Match skill names precisely

**Format:**
- Use standard section headings
- Avoid tables and columns
- Use bullet points
- Keep fonts simple (Arial, Calibri)
- Save as PDF or DOCX

**Content:**
- Start with strong summary
- Use action verbs
- Quantify achievements
- Include relevant keywords
- Tailor to each job

**Common Mistakes:**
- Using images or graphics
- Complex formatting
- Spelling errors
- Missing contact info
- Irrelevant information
- Too generic

## 💻 Technical Interview

### How It Works

The system extracts skills from your resume and generates relevant technical questions.

### Step-by-Step Guide

1. **Navigate to Technical**
   - Click "Technical" in the menu
   - See the upload interface

2. **Upload Resume**
   - Choose your resume file
   - System extracts skills automatically
   - Wait for processing

3. **Review Extracted Skills**
   - See list of detected skills
   - Verify accuracy
   - Note any missing skills

4. **Get Questions**
   - System generates skill-specific questions
   - Multiple difficulty levels
   - Covers all detected skills

5. **Practice Answering**
   - Read each question carefully
   - Think through your answer
   - Practice explaining concepts
   - Use examples from experience

## 🔧 Troubleshooting

### Camera Issues

**Camera not detected:**
- Check browser permissions
- Close other apps using camera
- Try different browser
- Restart computer

**Poor video quality:**
- Improve lighting
- Clean camera lens
- Check internet connection
- Reduce background apps

### Microphone Issues

**Not recording:**
- Check browser permissions
- Test microphone in settings
- Verify correct device selected
- Check volume levels

**Poor audio quality:**
- Reduce background noise
- Move closer to microphone
- Check microphone settings
- Use external microphone

### Upload Issues

**Resume not uploading:**
- Check file size (< 10MB)
- Verify file format (PDF/DOCX)
- Try different browser
- Check internet connection

**Processing fails:**
- Ensure text is selectable
- Avoid image-based PDFs
- Simplify formatting
- Try different file

### Performance Issues

**Slow loading:**
- Clear browser cache
- Close unnecessary tabs
- Check internet speed
- Restart browser

**Freezing/Crashing:**
- Update browser
- Disable extensions
- Clear cache and cookies
- Try incognito mode
